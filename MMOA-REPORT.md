### 3.1 **P2 หยั่งประเมิน — BARS-based Evaluation**

**ปัญหาเดิม:**
- ใช้ "ความรู้สึก" ให้คะแนน
- ไม่มีเกณฑ์ชัดเจน
- มีอคติ (halo effect, recency bias, etc.)

**Before:**
```
Rater ให้คะแนนตามความรู้สึก
    ↓
Accuracy: 60-70%
```

**After (Optimize with AI):**
```
BARS (Behaviorally Anchored Rating Scales)
+ AI ช่วยวิเคราะห์พฤติกรรมจริง
    ↓
Accuracy: 85-90%
```

**ตัวอย่าง BARS:**
```
CC1: ความเก่งในการนำผู้รับใช้ (Servant Leadership)

คะแนน 5 (ดีเยี่ยม):
  ✓ นำทีมแก้ปัญหาได้สำเร็จ 3 ครั้งขึ้นไป
  ✓ พนักงานในทีมพึงพอใจ ≥ 90%
  ✓ มีผลงานที่เป็นรูปธรรม 2+ โครงการ

คะแนน 3 (ดี):
  ✓ ทำงานตามคำสั่งได้ดี
  ✓ พนักงานในทีมพึงพอใจ 70-89%
  ✓ มีผลงาน 1 โครงการ

คะแนน 1 (ต้องปรับปรุง):
  ✗ ไม่รับฟังความคิดเห็น
  ✗ พนักงานในทีมพึงพอใจ < 70%
```
