## 🔥 หลัก MMOA Strategy

### **M**aximize — เพิ่มประสิทธิภาพสูงสุด
**แนวคิด:** ใช้ AI ช่วยทำงานหลายอย่างมากพร้อมกัน ลด bottleneck

| ปกติ (Manual) | Maximize (AI-Assisted) |
|---|---|
| HR คัดกรองใบสมัครทีละคน | AI คัดกรอง 10+ ช่องทางพร้อมกัน |
| สัมภาษณ์คนละ 1-2 ชม. | AI ช่วยสัมภาษณ์เบื้องต้น 15 นาที |
| ประเมินเครดิตประจำปี (manual) | AI รวบรวมข้อมูล 360° อัตโนมัติ |
| คำนวณเงินเดือน/สวัสดิการ manual | AI คำนวณและ export อัตโนมัติ |

#### ตัวอย่างใน Web PP7
- **P1 (แสวงหา):** AI คัดกรองใบสมัครจาก 10 ช่องทาง (Facebook, LinkedIn, JobThai, Indeed, ฯลฯ) → ลดเวลาจาก 8 ชม./วัน → 1 ชม./วัน
- **P2 (หยั่งประเมิน):** AI วิเคราะห์คำตอบจากแบบทดสอบ → สร้างคะแนน CC อัตโนมัติ
- **P6 (ค่าตอบแทน):** AI คำนวณเงินเดือน, สวัสดิการ, โอที → ลดเวลาจาก 3 วัน → 1 ชั่วโมง

---

### **M**inimize — ลดงานที่ไม่จำเป็น
**แนวคิด:** ตัดขั้นตอนซ้ำซ้อน, ลด human error

| ปัญหาเดิม | วิธีแก้ด้วย AI |
|---|---|
| คีย์ข้อมูล 2 ครั้ง (Excel → Web) | ใช้ API เชื่อมข้อมูลอัตโนมัติ |
| กรอง/จัดรูปแบบข้อมูล manual | AI จัดรูปแบบอัตโนมัติ |
| แจ้งเตือน/อนุมัติด้วยมือ | AI แจ้งเตือนอัตโนมัติ (Line/TG/Email) |
| สร้างรายงาน manual | AI สร้างอัตโนมัติทุกสัปดาห์ |

#### ตัวอย่างใน Web PP7
- **P3 (จับคู่):** ลดขั้นตอนการหาคนที่เหมาะกับงาน จาก 1 วัน → 5 นาที (AI จับคู่จากโปรไฟล์)
- **P5 (พัฒนา):** AI สร้างแผนพัฒนาอัตโนมัติตามผล P4 → ไม่ต้องคีย์แผนเอง
- **ทุก P:** ลดงานเอกสาร/รีพอร์ต ด้วย AI Dashboard

---

### **O**ptimize — ปรับให้เหมาะสม
**แนวคิด:** ใช้ data-driven, ปรับ flow ให้เร็วขึ้น

#### Optimized Flow
```
Before: 
  HR รับใบสมัคร → คัดกรอง 3 ครั้ง → สัมภาษณ์ → ตัดสินใจด้วยตัวเอง
  ⏱️ Time: 5-7 วัน
  📊 Accuracy: 60-70%

After (AI-Optimized):
  AI คัดกรองอัตโนมัต → Interview Score → แนะนำ candidate ที่เหมาะสม → HR ตัดสินใจ
  ⏱️ Time: 1-2 วัน
  📊 Accuracy: 85-90%
```

#### ตัวอย่างใน Web PP7
- **P4 (ประเมินผล):** ใช้ BARS (Behaviorally Anchored Rating Scales) → ลดอคติ, เพิ่มความเที่ยงตรง
- **ทุก P:** Data-driven decision → Dashboard แสดง KPIs แบบ real-time

---

### **A**utomate — ทำอัตโนมัติ
**แนวคิด:** ทุกงานที่ทำซ้ำๆ ให้ AI ทำอัตโนมัติ

| งาน Automate ได้ | ผลลัพธ์ |
|---|---|
| สร้างรายงานประจำสัปดาห์ | AI สร้างเองทุกเช้า 08:00 |
| แจ้งเตือนงานเลยกำหนด | AI แจ้งเตือนอัตโนมัติ (Line/TG/Email) |
| Export ข้อมูลออก/นำเข้า | AI จัดการข้อมูลอัตโนมัติ |
| คำนวณค่าตอบแทน/สวัสดิการ | AI คำนวณ + Export ไป BCT อัตโนมัติ |
| ติดตามสถานะงาน | AI ติดตาม + สรุปทุกเช้า |

#### ตัวอย่างใน Web PP7
- **ทุก P:** AI Dashboard แสดงสถานะแบบ real-time
- **P2 (หยั่งประเมิน):** AI คำนวณคะแนน CC อัตโนมัติ หลังทำแบบทดสอบ
- **P6 (ค่าตอบแทน):** AI คำนวณเงินเดือน 10 อัตโนมัติ + ส่ง BCT
- **ทุก P:** AI แจ้งเตือน Line/TG/Email เมื่อมีงานสำคัญ

## 🤖 YPNL Agentic Workflow

### AI Agents ที่ใช้ใน Web PP7

| Agent | หน้าที่ | ใช้ใน PP7 |
|---|---|---|
| **Y**OLO (Object Detection) | ดูภาพ/วิดีโอ/เอกสาร | P1: ตรวจสอบรูปใบสมัคร<br>P3: ประเมินเอกสาร<br>P4: วิเคราะห์พฤติกรรม (ถ้าต้องการ) |

## 📊 ตัวอย่างการใช้งานจริง

### P1: แสวงหา (Recruit)

**Before (Manual):**
1. HR รับใบสมัครจาก 10 ช่องทาง → ✅ Manual
2. คัดกรองเบื้องต้น → ✅ Manual (8 ชม./วัน)
3. นัดสัมภาษณ์ → ✅ Manual
4. สัมภาษณ์ → ✅ Manual
5. ตัดสินใจ → ✅ Manual

**After (AI-Assisted with MMOA):**
1. AI รวมใบสมัครจาก 10 ช่องทาง → ✅ **Automate** (n8n)
2. AI คัดกรองด้วย YOLO + LLMs → ✅ **Maximize** (0.5 ชม./วัน)
3. AI นัดสัมภาษณ์อัตโนมัติ (Line/TG) → ✅ **Automate** (n8n)
4. AI ช่วยสัมภาษณ์เบื้องต้น → ✅ **Optimize** (LLMs)
5. AI สรุปคะแนน CC → ✅ **Automate** (LLMs)
6. HR ตัดสินใจ → ✅ Data-driven

**ผลลัพธ์:**
- ⏱️ Time: 5-7 วัน → 1-2 วัน (60-70% faster)
- 📊 Accuracy: 70% → 85-90% (15-20% better)
- 💰 Cost: ลดงาน manual 80%

---

### P2: หยั่งประเมิน (Assess)

**Before (Manual):**
1. พิมพ์แบบทดสอบ → ✅ Manual
2. ผู้สมัครทำแบบทดสอบ → ✅ Manual
3. HR คำนวณคะแนน manual → ✅ Manual
4. HR สรุปผล → ✅ Manual
5. แจ้งผลให้ผู้สมัคร → ✅ Manual

**After (AI-Assisted with MMOA):**
1. AI สร้างแบบทดสอบตามประเภทใหม่/โอนย้าย/ฝึกงาน → ✅ **Automate** (LLMs)
2. ผู้สมัครทำแบบทดสอบใน Web → ✅ Standard
3. AI คำนวณคะแนน CC อัตโนมัติ → ✅ **Maximize** (LLMs)
4. AI สรุปผล + แนะนำ → ✅ **Optimize** (LLMs)
5. AI แจ้งผลอัตโนมัติ (Line/TG) → ✅ **Automate** (n8n)

**ผลลัพธ์:**
- ⏱️ Time: 2-3 วัน → 30 นาที (90% faster)
- 📊 Accuracy: 75% → 90% (15% better)
- 💰 Cost: ลดงาน manual 90%

---

### P4: ประเมินผล (Performance)

1. Rater ทำแบบประเมิน 360° → ✅ Manual
2. HR รวบรวมข้อมูล manual → ✅ Manual
3. HR คำนวณคะแนน → ✅ Manual
4. วิเคราะห์ feedback → ✅ Manual
5. สร้างรายงาน → ✅ Manual

**After (AI-Assisted with MMOA):**
1. Rater ทำแบบประเมินใน Web → ✅ Standard
2. AI รวบรวมข้อมูล 360° อัตโนมัติ → ✅ **Maximize** (n8n)
3. AI คำนวณคะแนน CC อัตโนมัติ → ✅ **Automate** (LLMs)
4. AI วิเคราะห์ feedback + แนะนำ → ✅ **Optimize** (LLMs)
5. AI สร้างรายงานอัตโนมัติ → ✅ **Automate** (LLMs)

**ผลลัพธ์:**
- ⏱️ Time: 2-3 สัปดาห์ → 2-3 วัน (80% faster)
- 📊 Accuracy: ไม่มีความเที่ยงตรง → BARS-based, data-driven (70% better)
- 💰 Cost: ลดงาน manual 85%

- 💰 Cost: ลดงาน manual 95%
