## Web PP7 Tab Switching Fix (Commit 0a47880)
- อ่าน index.html ทั้งหมด (~3,963 lines) เพื่อวิเคราะห์ root cause ที่ tab switching ไม่ทำงาน
- **ปัญหาหลักที่พบ:**
  1. CSS selector ว่าง `. {filter:blur(12px)!important;...}` ทำให้ CSS parse error
  2. `.tab-pane.hidden{display:none !important}` ขัดแย้งกับ Tailwind `hidden` class
  3. `showTab()` ใช้แค่ `classList.add/remove('hidden')` ไม่มี fallback `style.display`
  4. `classList.add(' ')` (class ว่าง) ในหลายจุด ทำให้ blur logic เสีย
- **สิ่งที่แก้:**
  1. เปลี่ยน CSS system: `.tab-pane{display:none}` + `.tab-pane.active-pane{display:block}` แทน reliance on `hidden`
  2. แก้ `showTab()` ให้ใช้ `active-pane` class + `style.display` fallback + `window.scrollTo({top:0})`
  3. เพิ่ม Event Delegation ที่ document level เป็น backup สำหรับ onclick inline
  4. เพิ่ม hash change listener สำหรับ URL hash navigation
  5. แก้ CSS selector ว่าง → `#app-wrapper.blurred{filter:blur(12px)...}`
  6. แก้ `classList.add(' ')` → `classList.add('blurred')` ทุกจุด
