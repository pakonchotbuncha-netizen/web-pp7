## Technical Decisions

- **CLI:** `/usr/local/bin/gog.real` (v0.14.0, build 469f4b4) — npm `gog` unreliable
- **Write strategy:** Single-cell writes per column to avoid pipe-separated parsing errors and row-splitting
- **Rate limits:** Google Sheets API quota exceeded frequently — need 30s+ delays between batch operations
- **Long text in S1:** Google Sheets API converts `\n` to new rows — must write as single compact string or use `;` separators. When text is too long, write row-by-row per column to avoid overflow.
