### INC Points System Design
```
BU บันทึก INC (รายคน/เป็นกลุ่ม) 
  → ระบบแปลงเป็น Points (1 บาท = 10 Points)
  → Points สะสมในบัญชีสมาชิก
  → สมาชิก Cashout ผ่าน MSP App ได้ทุกเมื่อ

ประเภท INC: sales, kpi, project, attendance, referral, innovation, customer_service, other
```

### Bulk Import Flow
```
BU หลายๆ คนกรอกข้อมูลพร้อมกัน ใน Google Sheets
  → Export CSV
  → Upload เข้า P6 Dashboard
  → Preview + Validate (Valid/Invalid แยกชัดเจน)
  → Import รายการที่ผ่าน
```
