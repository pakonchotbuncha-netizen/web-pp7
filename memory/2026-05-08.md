## Training Session — โครงสร้าง PP7 (บันทึก 2026-05-08 10:17-10:32 UTC)

ผู้ใช้ PKG-ปกรณ์(หนึ่ง) ให้คำนิยามถาวรของโครงสร้าง PP7 ทั้งหมด:

### Core Competency (CC) — แกนหลัก
- CC แบ่งเป็น 2 ด้าน:
  1. **ด้านทักษะ (Skills)** — สิ่งที่เรามองเห็นในความสามารถของสมาชิก
  2. **ด้านทุนองค์กร (Corporate Capital)** — ทัศนคติที่อยู่ในตัวคนที่เรามองไม่เห็น

### P1 — แสวงหา (Recruit)
- การค้นหาสมาชิกที่มี CC ตรงกับความต้องการขององค์กร

### P2 — หยั่งประเมิน (Assess)
- การหยั่งประเมินสมาชิกว่ามี CC ตรงกับเราหรือไม่
- สรุปสั้น: **"ประเมินก่อนรบ"**

### P3 — จับคู่คนกับงาน (Match)
- การจับคู่คนกับงาน โดยประเมินว่าผู้ที่ผ่าน P2 แล้ว เหมาะกับงานนี้หรือไม่
- เรียกอีกชื่อ: **"ประเมินก่อนทำงาน"**

### P4 — ประเมินผล (Performance)
- การประเมินหลังทำงาน ว่าทำงานไปแล้วควรพัฒนาและปรับปรุงเรื่องอะไร
- ประเมินตาม CC ของสมาชิก
- สรุปสั้น: **"ประเมินหลังรบ"**

### P5 — พัฒนา (Develop)
- ระบบพัฒนาสมาชิก ตาม CC (จากผลการประเมิน P4)

### P6 — ค่าตอบแทนและแรงจูงใจ (Welfare)
- ระบบค่าตอบแทนและแรงจูงใจ — การตอบแทนตาม CC ของสมาชิก

### P7 — คุณภาพชีวิต (Quality of Life)
- ไม่ใช่แค่สวัสดิการ แต่เป็นการสร้างสภาพแวดล้อมที่สมาชิก "อยากอยู่" ไม่ใช่แค่ "ถูกให้อยู่"
- 5 องค์ประกอบ:
  1. **สุขภาพ (Health)** — องค์กรใส่ใจทั้งกายและใจ
  2. **ความมั่งคั่ง (Wealth)** — ความมั่นคงทางการเงิน ไม่ใช่แค่เงินเดือน
  3. **บรรยากาศ (Well-being)** — ที่ทำงานที่ปลอดภัย เป็นตัวของตัวเอง ได้รับการยอมรับ
  4. **เวลา (Time)** — เวลาให้ครอบครัวและชีวิตส่วนตัว ไม่ใช่แค่เวลาทำงาน
  5. **อารมณ์ขัน (Fun)** — รอยยิ้ม ความผ่อนคลาย การทำงานที่มีความสุข

### ภาพรวม Flow PP7
```
P1 (แสวงหา) → P2 (ประเมินก่อนรบ) → P3 (จับคู่/ประเมินก่อนทำงาน)
    ↓
P4 (ประเมินหลังรบ) → P5 (พัฒนาตาม CC)
    ↓
P6 (ค่าตอบแทนตาม CC) + P7 (คุณภาพชีวิต 5 องค์ประกอบ)
```
