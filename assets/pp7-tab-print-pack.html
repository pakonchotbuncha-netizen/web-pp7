PP7 Tab Print Pack

ชุด review แยกทีละ tab สำหรับพิมพ์, เขียนกำกับ requirement, และส่งต่อให้นักออกแบบ

  • แต่ละ section แยกตาม 1 tab
  • มีพื้นที่หมายเหตุสำหรับเขียนว่าแต่ละ tab ต้องมีอะไรบ้าง
  • สามารถ print ทั้งไฟล์เป็นชุด review ได้ทันที

Tab 1, สรุปผู้บริหาร

เป้าหมาย: ดู KPI ภาพรวมและ executive dashboard

พื้นที่จด requirement

  • KPI สำคัญที่ต้องมี
  • กราฟ/การ์ดที่ต้องการเพิ่ม
  • ภาพที่อยากให้นักออกแบบเน้น

Tab 2, แสวงหา

เป้าหมาย: recruitment flow, vacancy, resignation bridge

พื้นที่จด requirement

  • vacancy ที่ต้องเห็น
  • status การสรรหาที่ต้องมี
  • filter / table / KPI ที่อยากเพิ่ม

Tab 3, หยั่งประเมิน

เป้าหมาย: assessment logic, readiness, interview queue

พื้นที่จด requirement

  • ขั้นตอน assessment ที่ต้องแสดง
  • เกณฑ์ readiness
  • ข้อมูลเชื่อมกับ matching / evaluation

Tab 4, จับคู่คนกับงาน

เป้าหมาย: matching matrix, role fit, downstream use

พื้นที่จด requirement

  • เกณฑ์ matching
  • matrix / score ที่อยากให้มี
  • จุดเชื่อมไป transfer / evaluation

Tab 5, ประเมินผล

เป้าหมาย: annual credit evaluation, model, formula, weights

พื้นที่จด requirement

  • สูตรที่ต้องรองรับ
  • weight config ที่อยากปรับได้
  • calculator / simulator / model selector

Tab 6, พัฒนา

เป้าหมาย: development plan, learning, coaching

พื้นที่จด requirement

  • development data
  • dashboard / learning path
  • จุดเชื่อมจาก evaluation

Tab 7, สวัสดิการ

เป้าหมาย: compensation, incentive, welfare

พื้นที่จด requirement

  • ข้อมูลค่าตอบแทน/แรงจูงใจที่ต้องมี
  • chart / summary ที่ต้องมี
  • จุดเชื่อมจาก annual credit

Tab 8, คุณภาพชีวิต

เป้าหมาย: wellbeing, quality of life, alert

พื้นที่จด requirement

  • มิติ quality of life
  • alert / risk ที่ต้องมี
  • ตาราง / KPI / chart

Tab 9, กฎหมายแรงงาน

เป้าหมาย: reference รายประเทศและ compliance view

พื้นที่จด requirement

  • ประเทศที่ต้องครอบคลุม
  • หัวข้อกฎหมายที่ต้องแสดง
  • รูปแบบการอ้างอิง

Tab 10, ระเบียบ PKG

เป้าหมาย: policy, internal rules, disciplinary guidance

พื้นที่จด requirement

  • หมวดระเบียบที่ต้องมี
  • เรื่องที่ต้องเน้น
  • การเชื่อมกับ workflow

Tab 11, Import / Export

เป้าหมาย: data exchange, template, validation

พื้นที่จด requirement

  • format ที่ต้องรองรับ
  • mapping / validation
  • สถานะ import-export ที่ต้องเห็น

Tab 12, System Map

เป้าหมาย: flow ภาพรวม PP7 ทั้งระบบ

พื้นที่จด requirement

  • flow ที่ต้องเน้น
  • node / relationship ที่ต้องมี
  • ข้อความสำหรับผู้บริหาร
