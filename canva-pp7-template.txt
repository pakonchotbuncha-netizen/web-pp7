🗺️ Layout — ภาพที่ 1: วงจร PP7

1. พื้นหลัง
- ใส่ rectangle เต็ม canvas → สี `#0B1120`
- เพิ่ม subtle gradient overlay: มุมบนซ้าย `#0B1120` → มุมล่างขวา `#1E1B4B`
- เพิ่ม subtle dot grid pattern (ใน Canva: Elements → Lines & Shapes → "Grid" → เลือกแบบจุดโปร่งใส)

2. หัวข้อด้านบน
- ตำแหน่ง: กลางบน, y ≈ 50px
- ข้อความ: `ระบบบริหารบุคลากร PP7 อัจฉริยะ — ขับเคลื่อนด้วย AI`
- ฟอนต์: Prompt Bold, ขนาด 42px, สี `#F8FAFC`
- เพิ่ม glow effect: Shadow → color `#3B82F6`, blur 20, offset 0

3. วง AI Core (ตรงกลาง)
- ตำแหน่ง: ศูนย์กลาง canvas (x=960, y=540)
- ใส่ Circle → ขนาด 280×280px
- สี: Gradient `#7C3AED` → `#3B82F6` (Linear, มุม 45°)
- เพิ่ม glow: Shadow → color `#7C3AED`, blur 40, spread 10
- ใส่ไอคอน AI Brain: Elements → search "brain icon" หรือ "AI icon" → เลือกแบบสีขาว/โปร่งแสง
- ใต้ไอคอน: ข้อความ `AI Core Engine` → Prompt SemiBold, 16px, สี `#F8FAFC`

4. วง Data Flow (วงนอก)
- ใส่ Circle → ขนาด 680×680px, อยู่ตรงกลาง
- สี: เส้นขอบ `#3B82F6` หนา 3px, ไม่มี fill (transparent)
- Opacity: 30%
- เพิ่ม glow: Shadow → color `#3B82F6`, blur 15

 P4 ประเมินผล | 1380, 760 (ขวาล่าง) | `#F59E0B` 
 P5 พัฒนา | 960, 900 (ล่างสุด) | `#EC4899` 
 P6 ค่าตอบแทน | 540, 760 (ซ้ายล่าง) | `#EF4444`

- กล่อง: Rectangle → ขนาด 220×180px, สี `#1E293B`, opacity 80%, border-radius 12px

เทคนิค Canva ที่สำคัญ

เพิ่ม Glow Effect
1. เลือก element → Effects → Shadow
2. ปรับ: Color = สีของ element, Blur = 15-40, Offset = 0, Spread = 5-10

Gradient Background
1. Elements → Gradient → เลือกแบบ Linear
2. ปรับสีและมุมตามต้องการ
3. หรือ: ใส่ 2 rectangle ทับกัน, rectangle บนทำ transparent gradient

ทำให้ข้อความอ่านง่ายบนพื้นเข้ม
- ใช้สี `#F8FAFC` (เกือบขาว) สำหรับหัวข้อ
- ใช้สี `#94A3B8` (เทาอ่อน) สำหรับรายละเอียด
- อย่าใช้สีเข้มบนพื้นเข้มเด็ดขาด

กลุ่ม (Group) element
- Ctrl+A (เลือกทั้งหมดในโมดูล) → Right-click → Group
- จะย้าย/ปรับขนาดได้พร้อมกัน

Export
- Share → Download → PNG (สำหรับพรีเซนต์) หรือ PDF (สำหรับพิมพ์)
- เลือก Quality: High
