## 🔷 ขอบเขตของ P2

P2 หยั่งประเมิน ครอบคลุมขั้นตอนที่ 8 ของ WI-002 ประกอบด้วย 4 กิจกรรมหลัก:
1. **สัมภาษณ์** — ประเมินความสามารถและทัศนคติ
2. **ทดสอบทักษะ** — วัดความรู้และทักษะตามตำแหน่งงาน
3. **CMC** — ประเมินความเหมาะสมเชิงวัฒนธรรมองค์กร
4. **ทดสอบทัศนคติ** — วัดทุนองค์กร (Corporate Capital)

## 🔄 S4 — กระบวนการใหม่

### P2 + P3 + ปิดใบสมัคร (สังเคราะห์จาก S2 + S3)

1. **AI สัมภาษณ์เบื้องต้น (Screening)** — คัดกรองคุณสมบัติพื้นฐานอัตโนมัติ
2. **AI ประเมินทักษะ** — ทดสอบความรู้/ทักษะตามตำแหน่ง
3. **AI ประเมิน CMC** — วัดความเหมาะสมวัฒนธรรมองค์กร
4. **AI ประเมินทัศนคติ** — วัดทุนองค์กร (Core Competency ตาม CC1-CC5)
5. **AI จับคู่คนกับงาน** — จากโปรไฟล์ผู้สมัคร + ตำแหน่งที่ว่าง
6. **AI ตรวจสอบประวัติ/เอกสาร** — เบื้องต้นอัตโนมัติ
7. **AI ปิดใบสมัครไม่ผ่าน** — อัตโนมัติเมื่อไม่ผ่านหลักเกณฑ์

## 🌐 S6 — ออกแบบ Web PP7 เพื่อรองรับ P2

### Database Schema
- ตารางข้อมูลผู้สมัคร (applicant)
- ตารางแบบสัมภาษณ์ (interview_form)
- ตารางผลทดสอบทักษะ (skill_test_result)
- ตารางผลทดสอบ CMC (cmc_result)
- ตารางผลทดสอบทัศนคติ (attitude_test_result)
- ตารางคะแนนสรุป (score_summary)
- ตารางประวัติการดำเนินการ (activity_log)
- ตารางเอกสารแนบ (attachments)

### UI Flow
- หน้าจัดตารางสัมภาษณ์ (จัดวัน/เวลา/ห้อง/คณะกรรมการ)
- หน้าแบบสัมภาษณ์ (สำหรับคณะกรรมการกรอกคะแนน)
- หน้าแบบทดสอบทักษะออนไลน์ (สำหรับผู้สมัคร)
- หน้าแบบทดสอบ CMC/ทัศนคติ (สำหรับผู้สมัคร)
- หน้าสรุปผลประเมิน (Dashboard กรรมการ)
- หน้าจับคู่คนกับงาน (AI จับคู่ + มนุษย์อนุมัติ)
- หน้าปิดใบสมัคร (ไม่ผ่าน/ส่งต่อหน่วยงานอื่น)

### Automation Rules
- Trigger: แจ้งเตือนผู้สมัครเมื่อถึงกำหนดสัมภาษณ์
- Trigger: แจ้งเตือนคณะกรรมการเมื่อต้องสัมภาษณ์
- Trigger: สรุปคะแนนอัตโนมัติเมื่อกรอกครบ
- Trigger: จับคู่งานอัตโนมัติจากคะแนน + โปรไฟล์
- Trigger: ปิดใบสมัครอัตโนมัติเมื่อคะแนนต่ำกว่าเกณฑ์

### Integration Point
- Line Notify / Telegram Bot (แจ้งเตือนผู้สมัคร)
- ระบบ SGM (ข้อมูลตำแหน่งงานว่าง)
- ระบบสมาชิก PKG (ข้อมูลอ้างอิง)
- ระบบตรวจสอบประวัติภายนอก

## 🔗 จุดเชื่อมต่อ (Integration)

### ก่อนเข้าสู่ P2
- **จาก P1 แสวงหา** → ผู้สมัครผ่านการคัดกรองใบสมัคร → ข้อมูลเข้าสู่ P2
- **จาก WI-004 โอนย้าย/เลื่อนตำแหน่ง** → สมาชิกภายในขอโอน/โปรโมท → เข้า P1 ใหม่ → ถึง P2

### หลัง P2 เสร็จ
- **ผ่านเกณ์** → ส่งเข้า **P3 จับคู่คนกับงาน**
- **ไม่ผ่าน** → ปิดใบสมัครอัตโนมัติ
- **เหมาะกับตำแหน่งอื่น** → ส่งต่อหน่วยงานอื่น
