## 📋 P2 — หยั่งประเมิน (0 เอกสารเฉพาะ)

**หมายเหตุ:** P2 เป็นส่วนหนึ่งของกระบวนการ P1 (ขั้นตอน 8-9)
- การสัมภาษณ์
- ทดสอบทักษะ
- CMC
- ทดสอบทัศนคติ

**ในทางปฏิบัติ:** P2 ถูกรวมอยู่ในเอกสาร P1 เป็นส่วนใหญ่ ยังไม่มีเอกสาร PM/WI/SD/FM แยกเฉพาะ

## 📋 P3 — จับคู่คนกับงาน (0 เอกสารเฉพาะ)

## 💡 ข้อสังเกต

1. **P2 และ P3 ยังไม่มีเอกสารแยก** — ถูกบูรณาการอยู่ใน P1
   - แนะนำ: สร้างเอกสาร PM/WI แยกสำหรับ P2 และ P3 โดยเฉพาะ
   - P2: คู่มือการประเมิน CC, แบบทดสอบทัศนคติ, เกณฑ์การสัมภาษณ์
   - P3: คู่มือการจับคู่คนกับงาน, เกณฑ์การจับคู่, แบบฟอร์มประเมินความเหมาะสม

2. **บาง WI ใน P4 เป็นเรื่องระบบ IT** ไม่ใช่การประเมินโดยตรง
   - WI-019, WI-020, WI-021: เกี่ยวข้องกับระบบ IT / สิทธิผู้ใช้
   - ควรย้ายไปอยู่ภายใต้หมวด IT Governance

3. **เอกสาร P1 มีจำนวนมากที่สุด** (15 เอกสาร) เพราะรวมถึงกระบวนการสรรหาทั้งหมด
