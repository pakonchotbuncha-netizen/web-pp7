# 📖 คู่มือ จป.เทคนิค

> สำหรับ จป.เทคนิค — อ่านแล้วรู้ว่าต้องตรวจวัดอะไร วัดอย่างไร

---

## 👋 คุณคือใคร?

คุณคือ **จป.เทคนิค** — ผู้ตรวจวัดและประเมินสภาพแวดล้อมการทำงาน คุณคือ "ตาและหู" ด้านความปลอดภัยขององค์กร

---

## 🎯 หน้าที่ของคุณ (สรุปสั้น ๆ)

1. **ตรวจวัดสภาพแวดล้อม** — แสง เสียง ความร้อน ฝุ่น สารเคมี
2. **ตรวจสอบเครื่องจักร** — จุดเสี่ยง อุปกรณ์ป้องกัน
3. **ตรวจระบบป้องกันไฟ** — ถังดับเพลิง Sprinkler Alarm
4. **วิเคราะห์ผล** — เปรียบเทียบกับมาตรฐาน
5. **รายงาน** — บอกผลและแนะนำวิธีแก้ไข

---

## 📚 สิ่งที่ต้องรู้

### 1. แสงสว่าง
| พื้นที่ | มาตรฐาน (lux) |
|--------|---------------|
| สำนักงาน | 300-500 |
| โรงงานทั่วไป | 200-300 |
| งานละเอียด | 500-1,000 |
| คลังสินค้า | 100-200 |

**เครื่องมือ:** Lux Meter
**วิธีวัด:** วางเครื่องที่ระดับทำงาน วัด 3 จุด取เฉลี่ย

### 2. เสียง
| ระดับ | ผล |
|------|----|
| < 80 dB(A) | ปกติ |
| 80-85 dB(A) | เริ่มต้องป้องกัน |
| > 85 dB(A) | **ต้องใส่ที่ป้องกันหู** |
| > 115 dB(A) | อันตรายทันที |

**เครื่องมือ:** Sound Level Meter, Dosimeter
**วิธีวัด:** วัด 8 ชั่วโมง (TWA) หรือวัดแบบ Area

### 3. ความร้อน
| งาน | WBGT สูงสุด |
|----|------------|
| งานเบา | 30°C |
| งานปานกลาง | 26.7°C |
| งานหนัก | 25°C |

**เครื่องมือ:** WBGT Meter
**วิธีวัด:** วัดในที่ทำงาน คำนวณ WBGT = 0.7WB + 0.2GT + 0.1DB

### 4. ฝุ่น
| ชนิด | PEL (mg/m³) |
|------|------------|
| ฝุ่นทั่วไป | 5 (หายใจได้), 10 (เข้าปอดไม่ได้) |
| ซิลิกา | 0.1 |
| ไม้ | 5 |

**เครื่องมือ:** Dust Sampler
**วิธีวัด:** เก็บตัวอย่างอากาศ ส่ง Lab

### 5. สารเคมี
- อ่าน **SDS** (Safety Data Sheet) ทุกชนิดที่ใช้
- ดูค่า: **TWA, STEL, IDLH**
- ตรวจสอบการเก็บรักษา: แยกประเภท มีป้าย ถาดรองรับ

---

## 🔧 วิธีตรวจสอบเครื่องจักร

### จุดเสี่ยงที่ต้องดู
1. **จุดหนีบ** (Pinch Point) — ส่วนที่เคลื่อนที่เข้าหากัน
2. **จุดหมุน** (Rotating Part) — เพลา เกียร์ ลูกกลิ้ง
3. **จุดตัด** (Shear Point) — มีด ใบตัด
4. **จุดกระแทก** (Impact Point) — ลูกตุ้ม ค้อน

### อุปกรณ์ป้องกันที่ต้องมี
- **Guard** — ที่กำบัง
- **Interlock** — หยุดเมื่อเปิด Guard
- **E-Stop** — ปุ่มหยุดฉุกเฉิน
- **LOTO** — Lockout/Tagout ก่อนซ่อมบำรุง

### LOTO (6 ขั้นตอน)
1. แจ้งผู้เกี่ยวข้อง
2. ปิดเครื่อง
3. ตัดพลังงาน (ไฟฟ้า ลม น้ำ)
4. ล็อค/ติดป้าย
5. ตรวจสอบว่าพลังงานหมดจริง
6. เริ่มซ่อมบำรุง

## 🔥 ระบบป้องกันอัคคีภัย

### ประเภทไฟและถังดับ
| ประเภท | เชื้อเพลิง | ถังดับที่ใช้ |
|--------|-----------|-------------|
| A | ไม้ กระดาษ ผ้า | น้ำ, โฟม, ABC |
| B | น้ำมัน แก๊ส | โฟม, CO₂, ABC |
| C | ไฟฟ้า | CO₂, ABC |
| K | น้ำมันทำอาหาร | Wet Chemical |

### ตรวจสอบทุกเดือน
- [ ] ถังดับเพลิง: ความดันครบ ใช้งานได้ตามกำหนด
- [ ] Sprinkler: ไม่ถูกบัง หัวฉีดไม่ตัน
- [ ] Alarm: ทดสอบทำงาน
- [ ] ทางหนีไฟ: เปิดได้ ไม่กีดขวาง
- [ ] ป้ายทางออก: สว่าง

## 🏥 ปฐมพยาบาลเบื้องต้น

### หลักการ
1. **ประเมินความปลอดภัย** — สถานที่ปลอดภัยไหม
2. **เรียกความช่วยเหลือ** — โทร 1669
3. **ปฐมพยาบาล** — ตามอาการ

### CPR (30:2)
1. กดหน้าอก 30 ครั้ง (ลึก 5-6 ซม. เร็ว 100-120 ครั้ง/นาที)
2. ผายปอด 2 ครั้ง
3. ทำซ้ำจนกว่า AED หรือรถพยาบาลมาถึง

### AED
1. เปิดเครื่อง
2. ติดแผ่นตามรูป
3. วิเคราะห์ — ห้ามแตะตัวผู้ป่วย
4. ช็อตถ้าเครื่องสั่ง
5. ทำ CPR ต่อ

## 💡 สิ่งที่ควรทำ
- ตรวจสอบ/เทียบเครื่องมือก่อนใช้ทุกครั้ง
- บันทึกผลทันที — อย่าจำ
- เปรียบเทียบกับมาตรฐานเสมอ
- แนะนำวิธีแก้ไขที่ปฏิบัติได้จริง

## 🚫 สิ่งที่ไม่ควรทำ
- ใช้เครื่องมือที่ยังไม่ได้เทียบ
- ประมาณค่า — ต้องวัดจริง
- รายงานแต่ค่าที่วัด โดยไม่แนะนำวิธีแก้
- ละเลยจุดเล็ก ๆ
