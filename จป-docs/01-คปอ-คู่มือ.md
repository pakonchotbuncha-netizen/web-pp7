# 📖 คู่มือ คปอ. — คณะกรรมการความปลอดภัย

> สำหรับ คปอ. ทุกท่าน — อ่านแล้วรู้ว่าต้องทำอะไร

---

## 👋 คุณคือใคร?

คุณคือ **คปอ.** (คณะกรรมการความปลอดภัย) — ตัวแทนของเพื่อนร่วมงานและนายจ้าง ที่มีหน้าที่ดูแลความปลอดภัยในองค์กร

---

## 🎯 หน้าที่ของคุณ (สรุปสั้น ๆ)

1. **เดินตรวจพื้นที่** — ดูว่าที่ทำงานปลอดภัยไหม
2. **รายงานอันตราย** — บอกนายจ้าง/จป. เมื่อพบสิ่งไม่ปลอดภัย
3. **รายงานอุบัติเหตุ** — เก็บข้อมูลเมื่อเกิดเหตุ
4. **เสนอแนะ** — แนะนำวิธีแก้ไข
5. **ประชุม คปอ.** — เข้าร่วมประชุมทุกเดือน

---

## 📚 สิ่งที่ต้องรู้

### 1. กฎหมายพื้นฐาน
- ทุกองค์กรต้องมี คปอ.
- คปอ. มีสิทธิตรวจสอบพื้นที่ ทำงานร่วมกับนายจ้าง
- ลูกจ้างมีสิทธิ **ปฏิเสธงานอันตราย** โดยไม่ถูกตอบโต้

### 2. วิธีตรวจพื้นที่
- เดินดูทุกจุดในพื้นที่ทำงาน
- ใช้ **Checklist** (ดูด้านล่าง)
- บันทึกสิ่งที่พบ แบ่งเป็น: ปลอดภัย / ไม่ปลอดภัย

### 3. วิธีรายงาน
- กรอกแบบฟอร์ม **สปท. 5-022** เมื่อเกิดอุบัติเหตุ
- รายงานเหตุเกือบเกิด (Near Miss) — เหตุที่เกือบเกิดอุบัติเหตุแต่ไม่มีผู้บาดเจ็บ

### 4. สิทธิลูกจ้าง
- ปฏิเสธงานที่อันตรายต่อชีวิตได้
- ร้องเรียนเรื่องความปลอดภัยได้
- ไม่ถูกไล่ออกเพราะร้องเรียน

---

## ✅ Checklist ตรวจพื้นที่ (ใช้ทุกเดือน)

### ไฟฟ้า
- [ ] สายไฟไม่ชำรุด
- [ ] ปลั๊กไฟไม่ overloaded
- [ ] มีเครื่องตัดไฟรั่ว (RCBO)

### เครื่องจักร
- [ ] มีที่กำบัง (Guard) ครบ
- [ ] ปุ่มหยุดฉุกเฉิน (E-Stop) ใช้งานได้
- [ ] ไม่มีเสียง/ความร้อนผิดปกติ

### ทางเดิน/ทางออก
- [ ] ทางเดินไม่กีดขวาง
- [ ] ทางหนีไฟเปิดได้ ไม่ล็อค
- [ ] ป้ายทางออกสว่าง

### PPE (อุปกรณ์ป้องกัน)
- [ ] พนักงานสวม PPE ถูกต้อง
- [ ] PPE เพียงพอและสภาพดี

### อัคคีภัย
- [ ] ถังดับเพลิงมีครบ ตรวจสอบแล้ว
- [ ] หัวฉีดน้ำ (sprinkler) ไม่ถูกบัง
- [ ] จุดรวมพลชัดเจน

### อื่น ๆ
- [ ] แสงสว่างเพียงพอ
- [ ] อากาศถ่ายเทดี
- [ ] พื้นไม่ลื่น/ไม่ขรุขระ
- [ ] เก็บสารเคมีถูกที่ มีป้าย

## 📅 ตารางงานของคุณ

| งาน | ความถี่ | เมื่อไหร่ |
|-----|--------|----------|
| ตรวจพื้นที่ | เดือนละ 1 ครั้ง | ตามกำหนด |
| รายงานผลตรวจ | ทุกครั้ง หลังตรวจ | ภายใน 3 วัน |
| ประชุม คปอ. | เดือนละ 1 ครั้ง | ตามนัด |

| รายงานประจำปี | ปีละ 1 ครั้ง | สิ้นปี |

## 💡 สิ่งที่ควรทำ
- เดินตรวจอย่างสม่ำเสมอ — อย่ารอให้สั่ง
- รายงานทุกสิ่งที่พบ — อย่าเงียบ
- พูดคุยกับเพื่อนร่วมงาน — ฟังปัญหา
- เสนอแนะด้วยข้อมูลจริง — ไม่ใช่ความรู้สึก

## 🚫 สิ่งที่ไม่ควรทำ
- เพิกเฉยต่ออันตราย
- รายงานเฉพาะเรื่องใหญ่
- กลัวที่จะรายงาน
- เก็บปัญหาไว้คนเดียว

---

## 📞 ติดต่อใคร?
- **จป.วิชาชีพ** — หัวหน้าทีมความปลอดภัย
- **หัวหน้างาน** — ผู้ดูแลแผนก
- **นายจ้าง** — ผู้มีอำนาจตัดสินใจ
