# 📖 คู่มือ จป.หัวหน้างาน

> สำหรับ จป.หัวหน้างาน — อ่านแล้วรู้ว่าต้องดูแลทีมอย่างไร

---

## 👋 คุณคือใคร?

คุณคือ **จป.หัวหน้างาน** — ผู้ดูแลความปลอดภัยในระดับหน่วยงาน/แผนก คุณคือ "ด่านหน้า" ที่ใกล้ชิดลูกน้องที่สุด

---

## 🎯 หน้าที่ของคุณ (สรุปสั้น ๆ)

1. **Toolbox Talk** — พูดเรื่องความปลอดภัยก่อนเริ่มงานทุกวัน
2. **ตรวจสอบ PPE** — ตรวจการสวมใส่ทุกวัน
3. **ทำ JSA** — วิเคราะห์ความปลอดภัยของงาน
4. **ทำ HIRARC** — ระบุอันตราย ประเมินความเสี่ยง
5. **สอบสวนอุบัติเหตุ** — หาสาเหตุรากฐาน
6. **สอนงาน** — พนักงานใหม่ + อบรมเฉพาะทาง

---

## 📚 สิ่งที่ต้องรู้

### 1. JSA (Job Safety Analysis)

**ทำไมต้องทำ?** — เพื่อวิเคราะห์อันตรายในแต่ละขั้นตอนของงาน **ก่อน** เริ่มทำ

**4 ขั้นตอน:**

**ขั้นที่ 1: เลือกงาน**
- งานใหม่ที่ยังไม่เคยทำ
- งานที่เปลี่ยนกระบวนการ
- งานที่เคยเกิดอุบัติเหตุ

**ขั้นที่ 2: แบ่งงานเป็นขั้นตอน**
- เขียนทีละขั้นตอน ตามลำดับที่ทำจริง
- เช่น งานเชื่อม: 1) เตรียมพื้นที่ 2) เตรียมเครื่องเชื่อม 3) ใส่ PPE 4) เชื่อม 5) เก็บงาน

**ขั้นที่ 3: ระบุอันตรายแต่ละขั้นตอน**
- แต่ละขั้นตอนมีอันตรายอะไร?
- เช่น ขั้นเชื่อม: รังสี UV, ควันเชื่อม, ไฟไหม้, ไฟดูด

**ขั้นที่ 4: กำหนดมาตรการควบคุม**
- จะป้องกันอันตรายอย่างไร?
- ใช้ Hierarchy of Controls (ดูด้านล่าง)

**ตัวอย่าง JSA Form:**

| งาน: __________________ | วันที่: ______ | ผู้ทำ JSA: ______ |
|------|------|------|------|------|
| ลำดับ | ขั้นตอน | อันตราย | มาตรการ | ผู้รับผิดชอบ |
| 1 | | | | |
| 2 | | | | |
| 3 | | | | |

### 2. HIRARC

**ทำไมต้องทำ?** — เพื่อประเมินความเสี่ยงและจัดลำดับความสำคัญ

**3 ขั้นตอน:**

**H = Hazard Identification (ระบุอันตราย)**
- กายภาพ: เสียง แสง ความร้อน ไฟฟ้า
- เคมี: สารเคมี ฝุ่น ไอระเหย
- ชีวภาพ: เชื้อโรค แบคทีเรีย
- การยศาสตร์: ท่าทาง น้ำหนัก ซ้ำซาก
- จิตสังคม: ความเครียด การกลั่นแกล้ง

**R = Risk Assessment (ประเมินความเสี่ยง)**

| ความรุนแรง ↓ / ความเป็นไปได้ → | แทบไม่เคย (1) | นานครั้ง (2) | บ่อย (3) | บ่อยมาก (4) |
|---|---|---|---|---|
| **เล็กน้อย (1)** | 1 | 2 | 3 | 4 |
| **ปานกลาง (2)** | 2 | 4 | 6 | 8 |
| **รุนแรง (3)** | 3 | 6 | 9 | 12 |
| **ร้ายแรง (4)** | 4 | 8 | 12 | 16 |

**ระดับความเสี่ยง:**
- 1-3 = **ต่ำ** (ยอมรับได้)
- 4-6 = **ปานกลาง** (ต้องควบคุม)
- 8-9 = **สูง** (ต้องควบคุมทันที)
- 12-16 = **วิกฤต** (หยุดงาน จนกว่าควบคุมได้)

**C = Risk Control (ควบคุม)**

**Hierarchy of Controls** (จากดีที่สุด → น้อยสุด):
1. **กำจัด** (Elimination) — ยกเลิกงานอันตราย
2. **แทนที่** (Substitution) — ใช้สารที่ปลอดภัยกว่า
3. **วิศวกรรม** (Engineering) — ติดตั้ง Guard, ระบบระบายอากาศ
4. **บริหาร** (Administrative) - อบรม หมุนเวียนงาน
5. **PPE** — อุปกรณ์ป้องกันส่วนบุคคล (สุดท้าย!)

### 3. การสอบสวนอุบัติเหตุ

**ทำไมต้องทำ?** — เพื่อหาสาเหตุที่แท้จริง ป้องกันไม่ให้เกิดขึ้นอีก

**5 Why Analysis:**
- ถาม "ทำไม?" ซ้ำ 5 ครั้ง (หรือจนกว่าจะเจอสาเหตุรากฐาน)

**ตัวอย่าง:**
1. ทำไมพนักงานนิ้วขาด? → เพราะมือเข้าไปในเครื่องจักร
2. ทำไมมือเข้าไป? → เพราะ Guard ถูกถอด
3. ทำไม Guard ถูกถอด? → เพราะทำงานไม่สะดวก
4. ทำไมไม่สะดวก? → เพราะ Guard ออกแบบไม่เหมาะกับงาน
5. ทำไมออกแบบไม่เหมาะ? → **เพราะไม่ได้ประเมินก่อนติดตั้ง** ← สาเหตุรากฐาน

**Fishbone Diagram (Ishikawa):**
```
               คน                    เครื่องจักร
               ↓                        ↓
         ┌─────┴─────┐          ┌───────┴───────┐
         │           │          │               │
    ─────┤           ├──────────┤               ├───── → อุบัติเหตุ
         │           │          │               │
         └─────┬─────┘          └───────┬───────┘
               ↑                        ↑
           วิธีทำงาน                สภาพแวดล้อม
```

**รายงานสอบสวนต้องมี:**
- วันที่/เวลา/สถานที่
- ผู้บาดเจ็บ/อาการ
- ลำดับเหตุการณ์
- สาเหตุโดยตรง
- สาเหตุรากฐาน
- มาตรการป้องกัน

### 4. Toolbox Talk

**คืออะไร?** — พูดคุยความปลอดภัยสั้น ๆ 5-10 นาทีก่อนเริ่มงาน

**รูปแบบ:**
1. **เปิด** — วันนี้จะพูดถึงเรื่องอะไร (1 นาที)
2. **เนื้อหา** — อธิบายสั้น ๆ พร้อมตัวอย่าง (3-5 นาที)
3. **ถาม-ตอบ** — ให้พนักงานถาม/แสดงความคิดเห็น (2-3 นาที)
4. **ปิด** — สรุป และเริ่มงาน (1 นาที)

**หัวข้อตัวอย่าง (หมุนเวียน):**
- สัปดาห์ 1: การสวม PPE
- สัปดาห์ 2: การยกของถูกวิธี
- สัปดาห์ 3: ไฟฟ้าปลอดภัย
- สัปดาห์ 4: การเก็บสารเคมี
- สัปดาห์ 5: ทางหนีไฟ
- สัปดาห์ 6: การใช้งานเครื่องจักร

**บันทึก:** ทุกครั้ง — วันที่ หัวข้อ ผู้เข้าร่วม

---

### 5. การสอนงาน

**Safety Induction (พนักงานใหม่):**
1. แนะนำนโยบายความปลอดภัย
2. พาดูพื้นที่ — จุดเสี่ยง ทางหนีไฟ จุดรวมพล
3. สอนการใช้ PPE
4. สอนขั้นตอนฉุกเฉิน
5. ลงชื่อรับทราบ

**ประเมินผล:** สอบถาม/ทดสอบหลังสอน

## 💡 สิ่งที่ควรทำ
- พูดเรื่องความปลอดภัยทุกวัน — ทำให้เป็นนิสัย
- ฟังลูกน้อง — พวกเขารู้ปัญหาดีที่สุด
- ทำ JSA ก่อนเริ่มงานใหม่เสมอ
- หาสาเหตุรากฐาน — อย่าแก้ที่อาการ
- ชมเชยเมื่อทำถูก — สร้างแรงจูงใจ

## 🚫 สิ่งที่ไม่ควรทำ
- ข้าม Toolbox Talk — "ไม่มีเวลา"
- เพิกเฉยเมื่อเห็นไม่สวม PPE
- หาว่าใครผิด — หาว่าระบบผิดตรงไหน
- ทำ JSA/HIRARC แล้วเก็บใส่แฟ้ม — ต้องใช้จริง
