## 📚 สิ่งที่ต้องรู้

### 1. ISO 45001 — Safety Management System (SMS)

**PDCA Cycle:**

**P = Plan (วางแผน)**
- นโยบายความปลอดภัย
- ระบุความเสี่ยงและโอกาส
- ระบุข้อกำหนดกฎหมาย
- ตั้งวัตถุประสงค์และแผน

**D = Do (ทำ)**
- โครงสร้างและความรับผิดชอบ
- สมรรถนะและการอบรม
- สื่อสาร
- ควบคุมปฏิบัติการ
- จัดการการเปลี่ยนแปลง
- จัดการผู้รับเหมา/ผู้ outsourced

**C = Check (ตรวจสอบ)**
- ตรวจวัดและประเมินผล
- การปฏิบัติตามกฎหมาย
- Internal Audit
- ทบทวนโดยผู้บริหาร

**A = Act (ปรับปรุง)**
- แก้ไขและป้องกัน
- ปรับปรุงอย่างต่อเนื่อง

---

### 2. สถิติความปลอดภัย

**LTIFR (Lost Time Injury Frequency Rate)**
```
LTIFR = (จำนวน LTIs × 1,000,000) / ชั่วโมงทำงานรวม
```
- LTI = อุบัติเหตุที่ทำให้หยุดงาน ≥ 1 วัน
- เป้าหมาย: ลดลงทุกปี

**SR (Severity Rate)**
```
SR = (จำนวนวันทำงานที่สูญเสีย × 1,000,000) / ชั่วโมงทำงานรวม
```

**AFR (Accident Frequency Rate)**
```
AFR = (จำนวนอุบัติเหตุ × 1,000,000) / ชั่วโมงทำงานรวม
```

**Leading vs Lagging Indicators:**
- **Lagging:** วัดผลหลังเกิดเหตุ (LTIFR, SR, จำนวนอุบัติเหตุ)
- **Leading:** วัดก่อนเกิดเหตุ (% ตรวจพื้นที่, % อบรม, Near Miss)

### 3. Internal Audit

**ทำไมต้องทำ?** — เพื่อตรวจสอบว่า SMS ทำงานตามที่ออกแบบ

**ขั้นตอน:**
1. **วางแผน** — กำหนดขอบเขต ตารางเวลา ทีมตรวจ
2. **เตรียม Checklist** — ตาม ISO 45001 ข้อ 4-10
3. **ตรวจ** — สัมภาษณ์ ตรวจเอกสาร ดูพื้นที่
4. **รายงาน** — Finding: Conformity, Minor NC, Major NC, OFI
5. **Follow-up** — ติดตามการแก้ไข

**ประเภท Finding:**
- **Conformity** — ทำถูกต้อง
- **Minor NC** — ผิดเล็กน้อย แก้ไขได้
- **Major NC** — ผิดร้ายแรง ระบบล้มเหลว
- **OFI** (Opportunity for Improvement) — แนะนำปรับปรุง

---

### 4. กฎหมายที่ต้องรู้

- พ.ร.บ. ความปลอดภัยฯ 2554
- กฎกระทรวง 2565 (มาตรฐานการบริหารจัดการ)
- ประกาศกรมสวัสดิการฯ เกี่ยวกับการจัด จป.
- กฎหมายเฉพาะ: โรงงาน, ก่อสร้าง, ขนส่ง, สารเคมี, เหมืองแร่

---

### 5. แผนฉุกเฉิน

**ประเภท:**
- อัคคีภัย
- สารเคมีรั่วไหล
- ภัยธรรมชาติ
- โรคระบาด

**ICS (Incident Command System):**
```
ผู้บัญชาการเหตุการณ์
        │
   ┌────┼────┐
   ↓    ↓    ↓
 ปฏิบัติ การวางแผน ส่งกำลัง
```

**ซ้อมแผน:**
- **Tabletop** — อภิปรายบนโต๊ะ
- **Functional** — ซ้อมบางส่วน
- **Full-scale** — ซ้อมเต็มรูปแบบ

## 💡 สิ่งที่ควรทำ
- ใช้ข้อมูลตัดสินใจ — สถิติ บอกแนวโน้ม
- นำเสนอผู้บริหารด้วยภาพรวม + แนวทางแก้ไข
- เชื่อมโยงความปลอดภัยกับธุรกิจ — ไม่ใช่แค่กฎ
- พัฒนาทีม จป. อย่างต่อเนื่อง

## 🚫 สิ่งที่ไม่ควรทำ
- รอให้เกิดเหตุใหญ่แล้วค่อยทำ
- ทำเอกสารอย่างเดียว — ต้องปฏิบัติจริง
- ซ่อนสถิติที่ไม่ดี — ต้องเปิดเผยเพื่อปรับปรุง
- ทำงานคนเดียว — ต้องทำงานเป็นทีม
